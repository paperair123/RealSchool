### 🏗️ **一、建筑类方块（模块化校园结构）**

> *核心目标：快速搭建多样化校园场景，支持自定义扩展*

1. **教室组件**
   - **方块**：
     - `黑板方块`（可书写文本，支持粉笔交互）
     - `组合式课桌`（单座/双座变种，抽屉可存储书本）
     - **讲台**（教师专属，触发上课事件）
   - **技术实现**：
     - 黑板使用`BlockEntity`存储文本数据，通过`@Inject`注入原版告示牌渲染逻辑
     - 课桌继承`AbstractFurnaceBlock`复用容器逻辑，减少开发量
2. **校园功能区**
   - **实验室**：`实验台`（合成化学药剂）、`显微镜`（观察粒子效果）
   - **图书馆**：`书架墙`（连接成整体）、`借阅台`（生成借书任务）
   - **体育馆**：`篮球架`（投掷检测）、`攀岩墙`（攀爬速度修正）
   - **技术要点**：
     - 篮球架通过`EntityProjectileHitEvent`监听投掷物命中
     - 攀岩墙使用`PlayerMoveCallback`事件修改爬梯速度
3. **室外景观**
   - **装饰**：`校名牌匾`（自定义校名）、`樱花树`（季节粒子效果）
   - **功能**：`自行车停放架`（实体自行车锚点）、`自动售货机`（快捷食物获取）

------

### 🧪 **二、功能性物品与设备**

> *增强校园生活模拟深度，驱动任务系统*

1. **学习工具**
   - **物品**：
     - `教材`（右键打开知识界面，记录学科进度）
     - `实验套装`（烧瓶+试剂，组合生成BUFF药水）
     - **乐器**：口琴（音阶交互）、三角铁（节奏小游戏）
   - **数据设计**：
     - 教材使用`NbtCompound`存储学习进度，同步至`PlayerDataStorage`
2. **校园用品**
   - **身份道具**：`学生证`（显示班级/学分）、`食堂饭卡`（交易代币）
   - **交互道具**：`扫帚`（清洁落叶粒子）、`扩音器`（范围通知NPC）

------

### 👨‍🏫 **三、实体与NPC系统**

> *构建动态校园生态，支持剧情事件*

1. **师生NPC**
   - **实体**：
     - `教师`（职业：教授/辅导员，发布课程任务）
     - `学生`（行为：抱书行走、课间聚群交谈）
   - **AI设计**：
     - 使用`FabricBrain`库实现日程表（上课→休息→社团活动）
     - 对话系统集成`VillagerOffer`实现任务交互
2. **特殊生物**
   - **吉祥物**：`校园守护灵`（被动生物，提供BUFF光环）
   - **事件实体**：`作弊幽灵`（考试期间出现，干扰玩家）

------

### ⚙️ **四、核心交互机制**

> *连接物品与场景，形成玩法闭环*

1. **教学系统**
   - **上课铃**：范围触发`EntityTeleportEvent`传送学生至教室
   - **考试机制**：
     - 课桌生成试卷（限时答题界面）
     - 根据成绩奖励`学分货币`
2. **社团活动**
   - **建造社**：提供`快速建造魔杖`（区域复制粘贴方块）
   - **音乐社**：多乐器合奏触发范围BUFF（如`速度II`）
3. **季节事件**
   - **开学典礼**：全校NPC聚集操场，玩家发表演讲（声望系统）
   - **校园祭**：解锁摊位小游戏（捞金鱼、射击挑战）

------

### 🧱 **五、资源与数据包规范**

> *确保资产兼容性与性能优化*

| 资源类型   | 路径规范                                      | 技术要求                   |
| :--------- | :-------------------------------------------- | :------------------------- |
| 方块材质   | `assets/modid/textures/block/blackboard.png`  | 尺寸≤512x512，禁用透明闪烁 |
| NPC模型    | `assets/modid/models/entity/teacher.geo.json` | 顶点数≤3000（VRAM优化）    |
| 音效       | `assets/modid/sounds/class_bell.ogg`          | 采样率44.1kHz，单声道      |
| 本地化文件 | `assets/modid/lang/zh_cn.json`                | UTF-8编码，键名全小写      |
