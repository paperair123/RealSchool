### 🏗️ **一、建筑类方块（模块化校园结构）**

> *核心目标：快速搭建多样化校园场景，支持自定义扩展*

#### 1. **基础矿物与材料**
   - **矿石**：
     - `石灰石 (Limestone)`：一种新的石头变种，常见于地表和山脉，是多种建筑材料的基础。
     - `石膏矿石 (Gypsum Ore)`：生成于地下，通常在Y=30到Y=80之间，开采后掉落2-4个`生石膏`。
     - `铝土矿 (Bauxite Ore)`：生成于丛林和热带草原生物群系，是提取`铝`的关键资源。
     - `铜镍矿 (Copper-Nickel Ore)`：与铜矿伴生，开采后掉落`镍矿石`和`铜矿石`，用于制作合金。
   - **基础材料**：
     - `生石膏 (Raw Gypsum)`：通过烧炼获得`熟石膏`。
     - `熟石膏 (Plaster of Paris)`：用于制作`粉笔`和`石膏板`。
     - `生石灰 (Quicklime)`：通过烧炼`石灰石`获得。
     - `熟石灰 (Slaked Lime)`：`生石灰`与水桶在工作台中合成，用于制作`粉笔`和`混凝土`。
     - `铝锭 (Aluminum Ingot)`：烧炼`铝土矿`获得，轻便且耐用。
     - `镍锭 (Nickel Ingot)`：烧炼`镍矿石`获得。
     - `不锈钢锭 (Stainless Steel Ingot)`：在`合金熔炉`中由铁、镍和少量煤炭合成，是制作高级设备的核心材料。

#### 2. **教室组件**
   - **方块**：
     - `黑板方块 (Blackboard)`：可书写文本，支持多种颜色的`粉笔`交互。
     - `组合式课桌 (Modular Desk)`：单座/双座变种，抽屉可存储物品。右击可坐下。
     - `讲台 (Podium)`：点击讲台可以触发上课事件，讲台前生成教师NPC，播放对应音频。
     - `粉笔块 (Chalk Block)`：9个`粉笔`合成，可作为白色装饰方块。
     - `石膏块 (Plaster Block)`：9个`熟石膏`合成，可作为白色装饰方块。
     - `投影仪 (Projector)`：放置在教室后方，右击可投射预设的图案（如世界地图、元素周期表）到`投影幕布`上。
     - `投影幕布 (Projector Screen)`：白色可悬挂方块，接收`投影仪`的投射。
     - `教室储物柜 (Classroom Locker)`：可堆叠的个人储物方块，每个玩家只能打开自己的储物柜。
   - **技术实现**：
     - 黑板使用`BlockEntity`存储文本和颜色数据。
     - 课桌可实现`Sit`接口，让玩家可以坐下。
     - 投影仪的投射效果可以通过在幕布前方生成一个特殊的、无碰撞箱的显示实体（Display Entity）来实现。

#### 3. **校园功能区**
   - **实验室**：
     - `实验台 (Lab Bench)`：用于合成化学药剂。
     - `显微镜 (Microscope)`：放入特定物品（如`植物细胞切片`）可观察放大的像素画。
     - `酒精灯 (Alcohol Lamp)`：提供热源，用于加速`化学反应器`的反应。
     - `通风橱 (Fume Hood)`：在其内部进行危险化学品合成时，可以避免负面药水效果。
   - **图书馆**：
     - `书架墙 (Bookshelf Wall)`：多个`书架`连接时，材质会自动拼接成一个整体。
     - `借阅台 (Lending Desk)`：生成借书任务，与图书管理员NPC交互。
     - `电子阅览桌 (Digital Reading Desk)`：内置电脑，可以查阅已解锁的模组内知识。
   - **体育馆**：
     - `篮球架 (Basketball Hoop)`：投掷`篮球`命中时触发红石信号并给予奖励。
     - `攀岩墙 (Climbing Wall)`：提供比梯子更快的攀爬速度。
     - `记分牌 (Scoreboard)`：可通过红石信号控制显示的数字。
     - `橡胶跑道 (Rubber Track)`：在其上行走会获得轻微的速度加成。
   - **食堂**：
     - `不锈钢餐盘架 (Stainless Steel Tray Rack)`：装饰性方块。
     - `打饭窗口 (Serving Window)`：与食堂大妈NPC交互，使用`食堂饭卡`购买食物。
     - `餐桌 (Canteen Table)`：大型桌子，可供多名玩家同时“使用”（坐下）。
   - **宿舍**：
     - `单人床 (Bunk Bed)`：功能与原版床类似，但占地更小，有上下铺变种。
     - `衣柜 (Wardrobe)`：大型储物容器，可存放盔甲和衣物。
     - `书桌 (Study Desk)`：带有台灯，提供一个明亮的光源。

#### 4. **室外景观与装饰**
   - **装饰**：
     - `校名牌匾 (School Nameplate)`：右击打开GUI，可输入文字自定义校名。
     - `樱花树 (Sakura Tree)`：新的树种，拥有粉色的树叶和独特的木头材质。
     - `公告栏 (Bulletin Board)`：可以贴上`纸`或`书`，供他人阅读。
     - `旗杆 (Flagpole)`：可以将各种颜色的`旗帜`挂在上面，旗帜会随风飘动。
   - **功能**：
     - `自行车停放架 (Bicycle Rack)`：可将`自行车`实体固定在上面。
     - `自动售货机 (Vending Machine)`：使用`硬币`购买各种饮料和零食。
     - `垃圾桶 (Trash Can)`：投入的物品会消失，可用于处理不需要的物品。
     - `路灯 (Street Lamp)`：提供不同风格的照明，如现代LED风格或复古风格。

### 🌾 **二、作物、食物与消耗品**

> *丰富校园生活细节，提供多样化的资源获取途径*

#### 1. **新增作物**
   - **`水稻 (Rice)`**
     - **种植**：只能种植在`水田方块`上。`水田方块`由在耕地上使用水桶制作。
     - **产物**：`稻谷`，可进一步加工成`大米`。
     - **实现**：创建一个新的耕地变种方块，并要求上方有水。
   - **`辣椒 (Chili)`**
     - **种植**：类似胡萝卜和马铃薯。
     - **产物**：`辣椒`，可作为烹饪调料。
   - **`茶叶 (Tea Leaf)`**
     - **种植**：一种灌木，成熟后可右键采摘`茶叶`，无需破坏植株。
     - **产物**：`茶叶`，用于制作各种茶饮。
   - **`咖啡豆 (Coffee Bean)`**
     - **种植**：在丛林木上生长的藤蔓状作物。
     - **产物**：`咖啡豆`，用于制作咖啡。

#### 2. **食物与饮料**
> *大部分食物需要在`厨房工作台`或`搅拌机`中制作*

   - **主食**：
     - `米饭 (Cooked Rice)`：`大米`在熔炉中烧制。
     - `盖浇饭 (Rice Bowl)`：`米饭` + `炒菜`。
     - `三明治 (Sandwich)`：面包 + 熟肉 + 生菜。
     - `汉堡 (Hamburger)`：面包 + 牛肉饼 + 生菜。
     - `披萨 (Pizza)`：需要`烤箱`制作，面团 + 番茄 + 奶酪 + 其他配料。
   - **菜肴**：
     - `炒时蔬 (Stir-fried Vegetables)`：胡萝卜 + 土豆 + `辣椒`。
     - `麻辣鸡块 (Spicy Fried Chicken)`：熟鸡肉 + `辣椒` + 面粉。
     - `番茄炒蛋 (Scrambled Eggs with Tomato)`：`番茄` + `鸡蛋`。
   - **零食**：
     - `薯片 (Potato Chips)`：切片的土豆在`油炸锅`中炸制。
     - `爆米花 (Popcorn)`：玉米在`微波炉`中加热。
     - `巧克力棒 (Chocolate Bar)`：可可豆 + 糖 + 牛奶。
   - **饮料**：
     - `奶茶 (Milk Tea)`：`茶叶` + 牛奶桶 + 糖。
     - `咖啡 (Coffee)`：`烤咖啡豆` + 热水。
     - `果汁 (Juice)`：任意水果 + `玻璃杯`在`搅拌机`中制作。
     - `可乐 (Cola)`：需要复杂的化学合成，在`化学反应器`中制作。

#### 3. **学习与功能性消耗品**
   - `教材 (Textbook)`：右键打开知识界面，记录学科进度。
   - `粉笔 (Chalk)`：用于在`黑板`上书写。
     - **配方**：2 `熟石膏` + 1 `熟石灰` = 8 `白色粉笔`。
     - **彩色粉笔**：`白色粉笔` + 任意染料。
   - `实验套装 (Experiment Kit)`：
     - `偷窥药水 (Peeking Potion)`：服用后为250m半径内的所有生物添加60秒发光效果，玩家自身无效果。
       - **配方**：在`实验台`上，使用`蒸馏水` + `荧光粉`（新物品，由荧光鱿鱼墨囊提炼） + `蜘蛛眼`。
     - `专注药水 (Focus Potion)`：服用后在10分钟内，通过学习获得的`学分`翻倍。
       - **配方**：`蒸馏水` + `青金石` + `咖啡豆`。
   - `篮球 (Basketball)`：可投掷，对实体造成少量伤害。
   - `乐器 (Musical Instruments)`：
     - `口琴 (Harmonica)`：右键吹奏，根据玩家朝向的俯仰角改变音高。
     - `三角铁 (Triangle)`：右键敲击，发出清脆的金属声。
   - `学生证 (Student ID)`：显示玩家姓名、班级、学分等信息。
   - `食堂饭卡 (Canteen Card)`：右键打开GUI，可存入`硬币`，用于在`打饭窗口`和`自动售货机`消费。

### ⚙️ **三、机器与交互设备**

> *驱动自动化生产与复杂物品合成，提供GUI交互*

#### 1. **`合金熔炉 (Alloy Smelter)`**
   - **功能**：用于制作`不锈钢锭`等合金材料。
   - **GUI**：拥有两个输入槽、一个燃料槽和一个输出槽。
   - **配方**：
     - `不锈钢锭`：1 `铁锭` + 1 `镍锭` + 1 `煤炭` -> 2 `不锈钢锭`。
   - **实现**：继承`AbstractFurnaceBlock`，修改配方逻辑和GUI纹理。

#### 2. **`厨房工作台 (Kitchen Counter)`**
   - **功能**：一个3x3的合成台，但专门用于制作食物。只有食物配方能在此合成。
   - **GUI**：类似原版工作台，但有独特的厨房主题背景。
   - **实现**：创建一个新的`RecipeType`，并将所有食物配方注册到该类型下。

#### 3. **`搅拌机 (Blender)`**
   - **功能**：制作各种果汁和混合饮料。
   - **GUI**：一个输入槽（用于水果等），一个液体槽（用于水或牛奶），一个输出槽。需要电力（RF/FE）或红石信号驱动。
   - **配方**：
     - `苹果汁`：1 `苹果` + 1 `水瓶` -> 1 `苹果汁`。
   - **实现**：使用`BlockEntity`处理物品和流体逻辑，并创建自定义GUI。

#### 4. **`化学反应器 (Chemical Reactor)`**
   - **功能**：用于进行复杂的化学合成，如制作`塑料`、`可乐`等。
   - **GUI**：拥有多个输入槽和流体槽，以及一个输出槽。可能需要`酒精灯`在下方加热才能工作。
   - **配方**：
     - `塑料颗粒 (Plastic Pellet)`：在`化学反应器`中，将`煤炭粉`和`蒸馏水`在加热条件下反应。
     - `塑料板 (Plastic Sheet)`：9个`塑料颗粒`合成。
   - **实现**：这是最复杂的机器之一，需要精细的`BlockEntity`逻辑来管理多种输入和反应条件。

#### 5. **`3D打印机 (3D Printer)`**
   - **功能**：消耗`塑料颗粒`和`染料`，打印出模组中的一些复杂装饰品或工具。
   - **GUI**：一个选择界面，用于选择要打印的物品，一个材料输入槽，一个染料槽。
   - **配方**：
     - `打印一个红色的篮球`：消耗 20 `塑料颗粒` + 2 `红色染料`。
   - **实现**：GUI中包含一个可滚动的列表，显示所有可打印的物品。`BlockEntity`负责处理打印过程和资源消耗。

### 👨‍🏫 **四、实体与NPC系统**

> *构建动态校园生态，支持剧情事件*

#### 1. **师生NPC**
   - **实体**：
     - `教师 (Teacher)`：职业多样，如`数学老师`、`化学老师`、`体育老师`。会根据课程表在不同功能区之间移动。
     - `学生 (Student)`：有不同的年级和班级，会根据上课铃声前往指定教室。
     - `图书管理员 (Librarian)`：在图书馆工作，提供借书还书任务。
     - `食堂大妈 (Canteen Staff)`：在食堂窗口工作，售卖食物。
   - **AI设计**：
     - 使用Fabric的AI API或`FabricBrain`库实现复杂的日程表（Schedule）和行为树（Behavior Tree）。
     - 例如，教师AI：`8:00`前往办公室 -> `8:30`前往教室 -> `9:15`返回办公室 -> `10:00`巡视走廊。

#### 2. **特殊生物**
   - **吉祥物**：`校园守护灵 (School Guardian)`：一只可爱的、可被驯服的生物，驯服后会跟随玩家，并提供小范围的增益效果（如“好运”）。
   - **事件实体**：`作弊幽灵 (Cheating Ghost)`：考试期间低概率在玩家附近生成，会发出干扰性的声音，并给予玩家短暂的“缓慢”和“挖掘疲劳”效果。使用`铅笔`（一种武器）可以快速驱散它。

### 🧱 **五、核心交互机制与配方示例**

> *连接物品与场景，形成玩法闭环*

#### 1. **教学与考试系统**
   - **上课铃**：一个方块，接收红石信号后会发出铃声。在铃声响起时，所有`学生`NPC会根据自己的课程表路径寻找到对应的教室座位。
   - **考试**：与`教师`NPC对话或在特定时间与`课桌`交互，打开一个GUI答题界面。题目可以预设在JSON文件中。答对题目会获得`学分`和`硬币`。
