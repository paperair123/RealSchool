这个完整包包含了 RealSchool 模组的所有文件和文档：

- **realschool-1.0.0.jar** - 编译好的模组文件（可直接安装）

### 💻 源代码
- **src/** - 完整的Java源代码

  - 实体系统
  - 世界生成
  - 数据生成
  - Mixin系统

### 📚 文档
- **README.md** - 项目说明文档
- **开发文档.md** - 详细的开发文档
- **RealSchool模组开发手记.md** - 开发过程记录

### 🔧 构建文件
- **build.gradle** - Gradle构建配置
- **gradle.properties** - Gradle属性配置
- **settings.gradle** - Gradle设置

## 🚀 快速开始

### 安装模组
1. 确保已安装 Minecraft 1.21.1 和 Fabric Loader
2. 将 `build/libs/realschool-1.0.0.jar` 复制到 `.minecraft/mods/` 目录
3. 启动游戏即可

### 开发环境
1. 确保已安装 Java 21 和 IntelliJ IDEA
2. 打开项目根目录
3. 运行 `./gradlew build` 构建项目
4. 运行 `./gradlew runClient` 启动测试客户端

## 🛠️ 开发说明

### 代码结构
```
src/main/java/papercliper/realschool/
├── RealSchool.java                 # 主模组类
├── RealSchoolClient.java           # 客户端初始化
├── block/                          # 方块相关
├── item/                           # 物品相关
├── entity/                         # 实体相关
├── world/                          # 世界生成
├── datagen/                        # 数据生成
└── mixin/                          # Mixin修改
```

### 构建命令
```bash
./gradlew build          # 构建模组
./gradlew runClient      # 运行测试客户端
./gradlew runServer      # 运行测试服务器
./gradlew runDatagen     # 生成数据文件
```
